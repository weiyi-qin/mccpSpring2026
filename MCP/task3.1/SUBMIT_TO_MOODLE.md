# How to Submit Task 3.1 to Moodle Forum

## Quick Guide

After generating your HTML visualizations, follow these steps to submit to Moodle.

## Step 1: Commit and Push to GitHub

```bash
# Navigate to your repository
cd /path/to/your/repo

# Add HTML files
git add visual.html visual_interconnection_text.html

# Commit
git commit -m "Task 3.1: Add paper structure visualizations"

# Push
git push origin main
```

## Step 2: Get Shareable Link

### Recommended: GitHub Pages

1. Go to your GitHub repo → **Settings** → **Pages**
2. Select **"Deploy from a branch"**
3. Choose **"main"** branch, **"/ (root)"** folder
4. Click **Save**
5. Wait 1-2 minutes
6. Your links will be:
   - `https://yourusername.github.io/repo-name/visual.html`
   - `https://yourusername.github.io/repo-name/visual_interconnection_text.html`

### Alternative: Raw GitHub Links

1. Go to your HTML file on GitHub
2. Click **"Raw"** button
3. Copy the URL
4. Use with HTMLPreview: `https://htmlpreview.github.io/?[your-raw-url]`

## Step 3: Reply to Moodle Forum

1. **Go to the forum post:**
   - https://buelearning.hkbu.edu.hk/mod/forum/discuss.php?d=345204
   - (Login required)

2. **Click "Reply"**

3. **Paste your reply:**
   ```
   Task 3.1 Submission
   
   Hierarchical Structure:
   https://yourusername.github.io/repo-name/visual.html
   
   Interconnected Network:
   https://yourusername.github.io/repo-name/visual_interconnection_text.html
   
   GitHub Repository:
   https://github.com/yourusername/repo-name
   ```

4. **Click "Post to forum"**
